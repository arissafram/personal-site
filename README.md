# Personal Site

## Technologies

React, TypeScript, Vite, CSS/SCSS, [react-jigsaw](https://github.com/arissafram/react-jigsaw) (GitHub package)

Node.js (v16 or higher), npm

## Run locally

```bash
git clone
npm install
npm run dev
```

## Build for prod

```bash
npm run build
```
